RAZ AI is an intelligent, customizable chatbot solution designed for modern web environments. Built from the ground up without relying on GPT or external AI APIs, RAZ AI leverages a smart pattern-recognition system, fuzzy matching logic, and integrated web features to create a fully interactive and evolving user experience.

ðŸ”§ Key Features

  • Custom JSON Knowledge Base: All answers are pulled from a self-managed JSON file for full control.
  • Advanced Question Recognition: Letter-by-letter fuzzy matching handles typos and word rearrangements.
  • Web Search Fallback: Searches online if no answer is found in the local dataset.
  • Image Recognition: Users can send images; the bot explains them with AI.
  • Secure Access: Login/register system with CAPTCHA protection.
  • User Profiles: Includes profile pictures and personal settings.
  • Dark Mode: Smooth theme toggle for better UX.
  • Auto-Correction & Math: Handles minor typos and evaluates expressions.
  • Review System: Fake reviews module for social proof simulation.
  • UI Polish: Clean layout, scroll animations, and responsive design.
